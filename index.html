Deb-b2g

Packaging b2g for debian based distributions.

Download as .zip Download as .tar.gz View on GitHub

Building your b2g package

Run |build.sh| to create a .deb package that includes b2g and a gaia checkout.

Once the package is installed, you can use b2g in two ways:

  1. Run /opt/b2g/launch.sh
  2. A new X session is also installed, that runs b2g in full screen. Just do a long press on the [Home] key to end the session.
